Introduction 
Selon l’Article deux (II), alinéas 3 de la Constitution : “Une proposition citoyenne peut être un projet de loi plus ou moins abouti ; un projet de modification d’une loi existante ; l’abrogation d’une loi existante ; un sujet de délibération ; une demande de révocation (II-11) ; une demande d’ajournement (II-12) ; un projet de création ou de dissolution d’ambassade (VI-3) ; un projet de traité (VI-5) ; une demande de délégation de l’ambassade (VI-7) ; une proposition de création de commission (VIII-5) ; une proposition de dissolution de commission (VIII-7) ; une proposition de réforme constitutionnelle (IX-1) ; ou une demande d’abolition constitutionnelle (IX-2). 
Par conséquent,  les proposition validé seront nommée par les abréviation suivante[a] : L pour loi[b] ; ML pour modification d’une loi [c]existante ; AL pour l’abrogation d’une loi existante ; D pour une délibération ; R pour une révocation de fonctionnaire ; Aj pour un ajournement ; CA pour une création d’ambassade; DA pour une dissolution d’ambassade ; T pour un traité; NDA pour une délégation de l’ambassade; CC pour une création de commission ; DC pour une dissolution de commission ; RC pour une réforme constitutionnelle ; ou AC pour une abolition constitutionnelle.
        En cas d’erreur, vous avez 4 jours à partir du jour de publication ou de re-publication pour la manifester. Les erreurs de numérotation seront rectifiées par des nombres à virgules.
